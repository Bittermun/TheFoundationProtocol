#### **TFP Client Core (tfp-client/lib/core/tfp_engine.py / Dart/TS)**
```python
# 1. Main public API – what the app exposes
class TFPClient:
    def request_content(root_hash: str, zkp_proof: Optional[bytes] = None) -> Content:
        # NDN Interest → RaptorQ shards → decode → lexicon reconstruct
        interest = ndn.create_interest(root_hash)
        shards = broadcast_receiver.receive(interest)  # or local mesh
        file_bytes = raptorq.decode(shards, k=original_symbols)
        return lexicon.reconstruct(file_bytes, on_device_ai_model)

    def submit_compute_task(task_recipe_hash: str) -> CreditReceipt:
        # User voluntarily joins marketplace
        task = compute_client.claim_task(task_recipe_hash)  # ZKP eligibility
        result_hash = execute_task_locally(task.shard)
        broadcast_result(result_hash)
        return credit_ledger.mint_credits(task.difficulty)

    def prove_access(root_hash: str, private_claim: bytes) -> ZKPProof:
        return zkp_prover.generate_proof(circuit="access_to_hash", private=private_claim)
```

#### **Broadcaster Service Endpoints (tfp-broadcaster/src/broadcaster.py)**

```python
# REST-like or NDN-based (deployed on tower / gateway)
@app.post("/seed")
async def seed_content(file: UploadFile, metadata: dict):
    root_hash = sha3(file.read())
    shards = raptorq.encode(file, redundancy=5%)
    multicast_transmit(shards, channel="ATSC3.0" or "5G-Broadcast")
    return {"root_hash": root_hash, "status": "broadcasting"}

@app.post("/broadcast_task")
async def broadcast_compute_task(recipe: TaskRecipe):
    # recipe = {task_type: "protein_fold", params_hash: "...", difficulty: 1000}
    shards = raptorq.encode(recipe.to_bytes())
    multicast_transmit(shards)
    return {"task_hash": recipe.hash, "status": "dripped"}
```

No RL in uplink routing is wise for predictability.

3 Simulation Test Cases (ns-3 + Mini-NDN)
Test Case 1: Shard Poisoning + Semantic Drift Attack

Topology: 1 broadcaster node + 50 edge nodes (Cortex-M4 class) in a star + mesh.
Injection: Attacker injects 20 % malicious shards with semantically altered recipes (e.g., fake physics constraints).
Expected: symbolic_preprocessor_validate() rejects → reconstruction blocked → <8 % failure rate.
Metric: Successful legitimate reconstructions vs. poisoned ones.

Test Case 2: Sybil Farm + PUF Identity Spoof

Topology: 200 Sybil nodes trying to claim compute tasks without valid PUF.
Injection: Fake RF fingerprints + replayed ZKP proofs.
Expected: puf_enclave_get_identity() + threshold VC rejects all Sybils.
Metric: Credit minting success rate for legitimate nodes ≥ 98 %; Sybil success = 0.

Test Case 3: Popularity Persistence + Asymmetric Uplink Under Congestion

Topology: 1 ATSC 3.0 broadcaster + 100 rural nodes + 1 gateway with variable 5G/Wi-Fi/LEO links.
Scenario: High-demand hash (education content) + low-demand hashes.
Injection: Network congestion + 30 % drop rate.
Expected: Demand-weighted credits correctly decay unused pins; uplink chooses lowest-cost 32-byte proof channel; ledger reconciles within 3 epochs.
Metric: Archive stability (high-popularity hashes remain cached) ≥ 95 %.
