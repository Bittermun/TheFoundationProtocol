### Bridge 3: Popularity→Persistence Economic Loop ✅

**Paths:**
- `tfp_client/lib/credit/`
- `tfp_client/lib/storage/`

| Module | Purpose | LOC |
|--------|---------|-----|
| `dwcc_calculator.py` | DWCC formula engine | ~280 |
| `hybrid_wallet.py` | Dual-balance wallet (compute + pinning) | ~320 |
| `pinning_manager.py` | Content pinning with decay | ~370 |

**Formula:**
```
DWCC = base_rate × (requests × storage_duration × semantic_value)
```

**Economic Model:**
- 50% compute/PoSI credits
- 50% archival pinning credits
- High-demand content earns pinning rewards
- Low-demand ages out naturally (decay)
- Network self-optimizes storage toward what's actually used

**Semantic Value Tiers:**
| Tier | Multiplier | Use Case |
|------|------------|----------|
| CRITICAL | 5.0× | Emergency info, public safety |
| HIGH | 3.0× | News, education, healthcare |
| MEDIUM | 1.5× | Entertainment, general info |
| LOW | 0.5× | Redundant, outdated content |
| DECAY | 0.1× | Near eviction threshold |

### Hierarchical Lexicon Tree (HLT) ✅

**Path:** `tfp_client/lib/lexicon/hlt/`

| Module | Purpose | LOC |
|--------|---------|-----|
| `tree.py` | Core tree structure (Root → Domain → Adapter) | ~300 |
| `delta.py` | Delta updates with precision anchors | ~200 |
| `sync.py` | Versioning, delta sync, tag-indexed discovery | ~250 |

**Structure:**
```
ROOT (base lexicon v1.0.0)
├── DOMAIN: medical (v2.1.0) [tags: healthcare, biology]
│   └── ADAPTER: medical_adapter_001 (v2.1.1) [precision_anchor: anchor_med_42]
├── DOMAIN: legal (v1.0.0) [tags: law]
│   └── ADAPTER: legal_adapter_001 (v1.0.1)
└── DOMAIN: technical (v3.0.0) [tags: engineering, code]
```

**Purpose:**
- Prevents semantic drift across devices
- Ensures same hash produces identical outputs
- Efficient delta updates (not full model transfers)
- Tag-indexed adapter discovery

## Usage Examples

### DWCC Calculator
```python
from tfp_client.lib.credit.dwcc_calculator import DWCCCalculator, SemanticValueTier

calc = DWCCCalculator(base_rate=1.0)

# Track content requests
calc.track_request("news_hash", SemanticValueTier.HIGH)
calc.track_request("news_hash")  # Popular content

# Process epoch and get rewards
rewards = calc.process_epoch(epoch_duration_hours=1.0)
# {'news_hash': 2.5}
```

### Template Assembly
```python
from tfp_client.lib.reconstruction.template_assembler import TemplateAssembler, Recipe
from tfp_client.lib.cache.chunk_store import ChunkStore
from tfp_client.lib.lexicon.hlt.tree import HierarchicalLexiconTree

chunk_store = ChunkStore()
hlt = HierarchicalLexiconTree()
assembler = TemplateAssembler(chunk_store, hlt)

# Create recipe
recipe = Recipe(
    content_hash="abc123",
    template_id="news_layout_v4",
    chunk_ids=["sky_42", "face_19", "text_delta"],
    ai_adapter="medical_v2.1"
)

# Assemble (checks cache first, only generates missing)
result = assembler.assemble(recipe)
print(f"Cached: {result.cached_chunks}, Missing: {result.missing_chunks}")
print(f"Bandwidth saved: {result.bandwidth_saved_bytes} bytes")
```

### Hybrid Wallet
```python
from tfp_client.lib.credit.hybrid_wallet import HybridWallet
from tfp_client.lib.credit.dwcc_calculator import SemanticValueTier

wallet = HybridWallet("node_001")

# Mint compute credits
proof_hash = b"compute_proof_32_bytes_here!!"
receipt = wallet.mint_compute_credits(100, proof_hash)

# Track content for DWCC rewards
wallet.track_content_request("popular_hash", SemanticValueTier.HIGH)

# Process DWCC epoch
earned = wallet.process_dwcc_epoch(epoch_hours=1.0)

# Spend credits
wallet.spend(50.0, credit_type="mixed")

# Check balance
balance = wallet.get_balance()
print(f"Compute: {balance.compute_credits}, Pinning: {balance.pinning_credits}")
```
